* Django forms - набор полей с данными определенного объекта.

* Используется для создания HTML-форм для ввода и далее отправки на сервер данных.

* Типов этих данных и целей использования django forms много.

* Формы создаются с помощью классов, которые наследуются от
forms.Form (тут сам пишешь форму вручную) либо forms.Modelform (тут попроще и меньше кода).

* Django Forms делает работу с формами на веб-сайте более простой и эффективной.
